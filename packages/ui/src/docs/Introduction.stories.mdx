# Perimetre UI

Welcome to the **Perimetre UI** component library — a brand-aware design system built on Tailwind CSS v4 and Radix UI primitives.

## Overview

This library provides a set of reusable React components that support visual polymorphism across multiple brands (Acorn, Sprig, Stelpro) through a three-tier design token system.

### Key Features

- **Brand Theming** — Switch between brands via the `data-pui-brand` attribute
- **Tailwind CSS v4** — All utilities use the `pui:` prefix to avoid conflicts
- **Radix UI Primitives** — Accessible, unstyled components as the foundation
- **CVA Variants** — Type-safe component variants with class variance authority
