Add dropdown-scoped semantic tokens for the FieldDropdown control: `color.dropdown.field-border`, `color.dropdown.field-item-hover-bg`, `color.dropdown.field-item-selected-bg`, and `shadow.dropdown-panel`. Acorn defaults mirror the existing input/dropdown tokens (no visual change). The OIQ Place pour toi brand overrides them to match Figma: neutral gray rest border (#e6e6e6), light-purple item hover (#f6e7ff) and selected (#efd3ff) fills, and a hard black offset panel shadow.
